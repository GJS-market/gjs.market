# GJS Market

**GJS Market** is a platform for discovering, sharing, and using presets, plugins, and templates built for GrapesJS.

🌐 Website: https://gjs.market

---

## 🚀 What is GJS Market?

GJS Market is a curated marketplace and discovery hub for the **GrapesJS ecosystem**.

It allows developers, designers, and SaaS builders to:
- Find ready-made templates and UI blocks
- Discover plugins and extensions
- Share their own presets with the community
- Accelerate development of page builders and document editors

---

## 🧩 What You Can Find

### 1. Presets
Pre-configured GrapesJS setups with:
- Custom UI
- Styled components
- Prebuilt layouts

### 2. Plugins
Extend GrapesJS functionality:
- Custom components
- Editor enhancements
- Integrations (CRM, APIs, etc.)

### 3. Templates
Ready-to-use designs:
- Landing pages
- Email templates
- Document layouts (PDF-like editors)

---

## 🎯 Who is it for?

GJS Market is designed for:

- **Frontend developers** building editors with GrapesJS
- **SaaS founders** creating tools like:
  - Website builders
  - Email editors
  - Document/PDF generators
- **Agencies** that need reusable layouts
- **Indie hackers** looking to ship faster
